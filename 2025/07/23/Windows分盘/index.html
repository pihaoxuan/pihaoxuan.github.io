Windows分盘

发表于 2025-07-23 分类于 Windows 阅读次数： Disqus：
本文字数： 888 阅读时长 ≈ 1 分钟

WIindows不会分盘怎么办？在线等急急急！

由于Windows将C盘作为系统盘，所以C盘的容量非常重要，如果C盘容量不足，可能会导致系统运行缓慢，甚至无法启动。因此，将其他盘符作为数据盘是一个很好的选择。下面介绍Windows分盘的方法。

前置条件：C盘容量足够大，能够将空间分出去。建议预留给C盘至少100G。

在桌面找到“此电脑”（如果桌面上没有，按WIndows+E打开后在左侧找到也是一样的），右键点击“此电脑”，选择“管理”（如果没有管理，先点击“展示更多选项”，就可以找到了），在管理中选择“磁盘管理”。出现如下界面：

其中的小方格就是你的这块磁盘被分尸的碎尸块。其中展示给用户的部分是有显著标志的，例如上图中的Windows-SSD（C:）就是C盘。其它小块的可以暂时不用管，是系统为了分盘而产生的内存块。

1. 分区

  1. 在上述界面，找到需要分区的磁盘，右键点击，选择“压缩卷”。顾名思义，就是将你选择的这块空间中空闲空间进行压缩。填写你想要分出的空间大小。这一步一般是用MB为单位，例如我想要分出100G，就填写102400。但是实际上由于格式化的格式需要占用空间，所以实际分出的空间会略小于100G。输入完需要分出的空间大小后，点击“压缩”，等待完成。
  2. 完成后，会多出一个空闲空间，如下图：

    右键点击，选择“新建简单卷”。因为是将分出的所有空间新建一个盘，所以输入简单卷的大小按照最大的输入。

    输入完后，点击下一步，只需要选择一个好听的名字就好了（其实也就只能选字母），左侧的选项不需要更改。
  3. 接下来一直点下一步就行，不需要有额外的操作，知道最后点完成，新建卷就完成了。

2. 合并

合并有两个非常重要的前置条件：
①需要有一个已经格式化的卷（被合并的卷），里面不能有任何数据，所以在合并卷前请做好数据备份！！
②两个卷必须相邻，但无所谓左相邻还是右相邻

  1. 在磁盘工具中，找到需要被合并的卷，右键，选择删除卷。如下图：

    删除后会变成未被分别的灰色。
  2. 选择需要合并的卷，右键，选择扩展卷。如下图：

    点击后会出现选择卷的界面，确保被合并的卷在右侧的已选择方框内即可。然后点击“下一步”，完成即可。

END
