Windows电脑新机注意事项

发表于 2025-07-23 更新于 2025-08-19 分类于 Windows 阅读次数： Disqus：
本文字数： 3.2k 阅读时长 ≈ 3 分钟

震惊！拿到新电脑后应该要做这些事情！

验机

  1. 拿到快递先开始拍视频，再拆快递，以防商家或快递推卸责任。【后面有兴趣剪辑成开箱Vlog也方便，但是需要选择好机位】
  2. 开箱后先检查电脑主体和配件是否缺失
  3. 检查无误后，检查电脑ABCD面【A面：顶部、B面：屏幕、C面：键盘、D面：底部】是否有划痕等瑕疵。
  4. 视频录制结束

开机

  1. 开始录视频
  2. 插上电源、按电源键开机
  3. 选择地区，输入法等，第二种键盘布局暂时可以跳过。
  4. Windows11开始强制用户连网，但是联网系统自动激活不支持7天无理由退货。如果不想连网请看第五步，如果不想折腾请看第六步。
  5. 不想连网。按住shift+F10会弹出cmd框【也就是命令行界面，装x用的】。如果没有反应，就按住Fn+shift+F10。然后输入以下代码，输入完后按回车即可： 回车后电脑应该会立即重启，然后重新进入选地区、输入法这些步骤，但是到了连网这一步，就会有我没有Internet连接字样，点击即可。点击后会再有一层提示，问是否进入受限设置。【受限设置听起来很吓人，其实就是没有网络，有一些设置需要网络连接才能设置，之后连网可以自行设置】
  6. 想连网。那你就连吧
  7. 输入用户名，建议英文，尤其是需要写代码的同学。输入密码，即锁屏密码
  8. 隐私设置，建议将最后的广告ID关掉，其他保持默认。当然想自己琢磨着关掉也行，都不影响。
  9. 完美结束，进入桌面

硬件检查

  1. 检查CPU、内存、磁盘信息。按住ctrl+shift+ESC进入任务管理器。点击左侧性能，能够看到CPU、内存、磁盘的信息。注意对比CPU是否和购买时所看的CPU一致。内存指的是运行空间，并不是存储空间。磁盘指的是存储空间。例如16G + 1T则内存为16G，磁盘空间为1T。如下图（从上到下分别是CPU信息、内存信息、磁盘信息）：



    ⚠我这里有两块磁盘才会显示两个disk，新机一般只有一块磁盘。右边红色框中可以看磁盘的容量。因为需要格式化，所以一般会比标注容量小。例如我这是一块512G的固态，但是实际容量为477G。如果是1T的固态，实际容量大概为931G。
    事实上，这个界面也可以查看GPU的信息。如下图：

    但是第二步中的更为详细。
  2. 检查GPU信息。上一步的界面可以关闭。按下Windows，等待Windows菜单弹出，输入Nvidia后，打开出现的Nvidia Control Panel。

    打开后点击左下角的系统信息，就能出现非常详细的GPU信息，例如GPU的名称、驱动版本号、驱动类型、CUDA内核、最大功率等等信息，以后有需要也可以在这里查看。

  3. 查看磁盘分区，判断是否需要人为分区。按住Windows+E，点击左侧的此电脑，此时会出现C盘。如果整个电脑只有C盘，则可以考虑分盘。如果是其他情况，也可以根据自己的喜好进行合并划分处理。具体操作请看Windows分盘

软件下载和安装【以火绒为例】

  1. 不建议在电脑自带的商城下载，因为会把软件下在C盘【并非下到C盘不好，而是默认的数据也会存储到C盘，久而久之，C盘容易红】。
  2. 建议前往浏览器查找需要下载的软件，然后下载exe安装包。注意辨别非官方网站，一般看网址分辨。不会的可以往下拉看TIPS1。
  3. 分盘后可以在磁盘中划出一部分空间专门用于软件安装和存放软件产生的数据。也可以直接新建一个文件夹存放。
  4. 在第三步中的位置新建文件夹，装什么软件就叫什么名字，建议英文名，不会就拼音也行。
  5. 在浏览器中下载的软件，默认是存放在下载文件夹中，可以直接从浏览器直达该文件夹。不会的看TIPS2。
  6. 双击后缀为exe的可执行文件，进入软件安装步骤。没有后缀的看TIPS3。后缀是.后面的内容，例如.exe。
  7. 出现软件安装的界面不要立马点类似极速安装、一键安装等字样，都是骗小孩的。要先找到更多安装选项，找到安装目录，将安装目录改成第四步中新建的文件夹，这样才真正将软件安装到了你给软件划分的牢笼了。否则就是安装到了C盘，久而久之电脑容易卡。
  8. 完成上一步之后，就可以无脑点击安装界面的下一步了。直到安装完成。

TIPS1 辨别网站是否官方

没有通用方法，全靠经验。不过官方网站有些比较明显的特征。
1️⃣ 例如火绒安全官网（下图中第一个）的网址是：https://www.huorong.cn/
网址的内容即软件名称的拼音，百度也是这样的。而有一个盗版网站（下图中最后一个）https://www.mydown.com/soft/357/472030857.shtml可以点进去看看虽然也是下载这个，但是不能保证官方正版，甚至不能保证纯净无毒，网站叫极速下载，一看就是下三滥的网站。

2️⃣ 网页内容看起来很高级的，一般就是官网。因为没有盗版网站会给你精心设计UI，除非想骗你钱。
3️⃣ 网页内没有广告。

另外国内环境建议使用Bing搜索引擎进行搜索，众所周知，百度靠投放广告赚钱。如果有条件，也可以使用Google进行搜索，Bing和Google搜索出来的内容会纯净很多。该设置项在Windows自带的浏览器Edge中可以找到。Bing也是微软自家的搜索引擎。

TIPS2 浏览器直达下载文件夹

点击下载后，右上角会有个下载的动画，抓住机会，鼠标悬浮在下载项右边，有个文件夹的图标，点开就能打开该下载项所在的文件夹。
如果没抓住机会，也可以在浏览器界面按住Ctrl+J打开下载界面。

TIPS3 打开文件后缀

按住Windows+E打开文件资源管理器，找到工具栏中的视图，点击视图，找到最下面的展示，点击展示后，可以看到有个文件扩展名，勾选上就能显示文件后缀了。

TIPS4 关于软件是否需要下载到C盘

网上很多说法是将软件盘和系统盘区分，例如系统盘在C盘，则将软件全部放入D盘，将软件和系统区分开。原因是系统盘东西存多了会影响电脑的运行速度。但是事实上，抛开剂量谈毒性都是耍流氓。上述说法也一样，如果系统盘变红了，那无论软件是否存放在系统盘，都会影响电脑的运行速度；换言之，如果C盘没有变红，那软件下载到C盘反而在一定程度上能够加快电脑系统对软件的读取，从而加快软件的运行速度。所以，问题的根源在于在初始时，应该给C盘划分多少空间，而非在意软件是否安装在C盘。
但是，许多软件默认存放数据的地方也就是安装的地方。例如，将微信安装在C盘后，那么微信的聊天记录，所看过的图片视频也全部存放在C盘，这无疑会加剧C盘红温的速度，所以，可以将微信的数据存放在C盘以外的硬盘。
值得注意的是，当前新电脑基本默认一块物理硬盘（就是拆开来只有一块存数据的硬盘），分盘只是逻辑上划分成两块，实际上还是只是存放在那一块物理硬盘上。所以，分盘只是为了方便管理，并不是为了提高速度。当然，如果自己加装了一块盘，那么就确实能够分开存储。
再者，如果后续可能会重装系统，则无论是否有多块物理硬盘，都可以将数据存储在系统盘以外的盘。这样重装系统时，只需要格式化系统盘，即可完成重装，其他盘的数据不会丢失。例如，如果只有一块物理硬盘，可以将其划分为CD盘，C盘为系统盘，D盘为数据盘。当把微信的聊天数据存放在D盘时，即便微信安装在C盘，电脑C盘格式化后，将微信重新下载安装，仍然可以读取D盘中的聊天数据。
综上所述，分盘可以根据需求为之。

软件建议

建议下载：
  1. 火绒安全
  2. Chrome
  3. Steam（学习用）
  4. 原神（锻炼用）
  5. BandZip（这个即使是官网也很丑）
  6. everything（电脑全局文件搜索，比Windows自带的搜索快无数倍）
  7. snipaste（截图软件，不需要微信QQ也能截屏，而且速度快功能多）
不建议下载：
  1. 搜狗输入法（有广告，用微软自带的输入法就行，喜欢皮肤那就得忍受广告）
  2. 360安全卫士（有广告，而且会占用大量内存，建议使用火绒，其实360全家桶都不建议碰，只要碰一个，搞不好就会强制安装所有）
  3. 鲁大师（有广告，轻易卸不掉，建议使用火绒）
